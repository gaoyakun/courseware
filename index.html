  按照某种规则，将计算机内一组无序的数据调整为有序，这种操作我们称之为“排序”。

未排序

已排序

  1. 大量的计算机应用程序都有对于数据排序的需求。
  2. 许多复杂的算法当中都对有序数据的要求，或者把排序作为算法中的一个必要环节。
  3. 各种不同的基础排序算法中分别涵盖了很多技术，并可以引申到更多的算法领域当中。
  1. 按照待排序数据的存储方式可分为内部排序和外部排序。
  2. 按照稳定性可分为稳定排序和不稳定排序。
  3. 按照排序方式可分为基于比较的排序和非基于比较的排序。

void bubbleSort (int numbers[], int count) {
    for (int i = count - 1; i >= 0; i--)
        for (int j = 0; j < i; j++) {
            if (numbers[j] > numbers[j+1]) {
                int tmp = numbers[j];
                numbers[j] = numbers[j+1];
                numbers[j+1] = tmp;
            }
        }
}
